Project Overview

The Plane Ticket Booking System is a Java-based console application that allows users to book, cancel, and view flight tickets efficiently. It includes seat selection, passenger details, and ticket confirmation.

This project demonstrates object-oriented programming concepts like classes, methods, arrays, and file handling. It also covers input validation, menu-driven navigation, and basic data persistence.

Key Features

Ticket Booking

Users can book tickets by providing passenger details and selecting seats.

Ticket Cancellation

Allows users to cancel booked tickets with proper updates in the system.

Passenger Management

Stores and displays passenger information, ensuring smooth management.

File Handling

Maintains records using file storage for persistence between sessions.
