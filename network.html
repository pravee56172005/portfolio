Project Overview

This project implements Network Flow algorithms in Java, focusing on solving the Maximum Flow problem in directed graphs. It was designed to handle real-world scenarios such as traffic routing, resource allocation, and bipartite matching.

The application supports both Ford-Fulkerson and Edmonds-Karp algorithms, with time complexity analysis and step-by-step visual representation of augmenting paths.

Key Features

Graph Representation

Efficient adjacency list structure for storing network graphs.

Max Flow Calculation

Implements Ford-Fulkerson and Edmonds-Karp algorithms in Java.

Performance Analysis

Compares time complexity between different approaches.

Educational Tool

Helps students and researchers understand flow algorithms step-by-step.

// Ford-Fulkerson Method
boolean bfs(int s, int t, int[] parent) {
    boolean[] visited = new boolean[V];
    Queue q = new LinkedList<>();
    q.add(s);
    visited[s] = true;
    parent[s] = -1;

    while (!q.isEmpty()) {
        int u = q.poll();
        for (int v = 0; v < V; v++) {
            if (!visited[v] && residual[u][v] > 0) {
                q.add(v);
                parent[v] = u;
                visited[v] = true;
            }
        }
    }
    return visited[t];
