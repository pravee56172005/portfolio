Project Overview

The Dice Game is a simple web-based game developed using HTML, CSS, and JavaScript. Players roll two dice and compete to reach the target score first. The game includes player turns, score tracking, and basic game logic for a fun interactive experience.

The project demonstrates my ability to work with DOM manipulation, event handling, and simple game logic. The interface is clean, responsive, and easy to play.

Key Features

Interactive Gameplay

Players can roll dice and see results in real-time with animations.

Score Tracking

The game tracks player scores and displays the winner when the target score is reached.

Responsive Design

Works smoothly on desktop and mobile devices, adapting to screen size.

Lightweight & Fast

No heavy frameworks; the game loads quickly and runs efficiently.
