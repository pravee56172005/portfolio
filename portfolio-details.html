Project Overview

Urban Wear is a modern e-commerce website for clothing retail, designed to provide users with a seamless online shopping experience. I implemented the frontend, styled the product catalog, shopping cart, and checkout pages, while also integrating backend APIs for product management and payment simulation.

The website features responsive design for mobile and desktop, intuitive navigation, and visually appealing product displays to enhance user engagement.

Key Features

Responsive Design

Fully responsive layout for mobile, tablet, and desktop users.

Shopping Cart

Interactive cart system allowing easy addition/removal of products.

Checkout & Payment

Secure checkout flow with simulated payment integration.

Backend Integration

Connected frontend to backend APIs for product and order management.
